# Calendario Gotas

A small, client-side calendar web app for tracking scheduled eye-drop (gotas) reminders. This repo contains a minimal HTML/CSS/JS project intended to run locally in the browser.

**Features**

- Simple calendar UI (monthly view)
- Add and view reminders for eye-drop schedules
- Lightweight: single-page app with no backend

**Project structure**

- `index.html` — main page and app entry
- `assets/styles.css` — styling for the UI
- `assets/scripts.js` — application logic and event handlers

Getting started

1. Clone or download the repository to your machine.
2. Open `index.html` in your browser (double-click or use "Open File").

No build steps or dependencies are required.

Usage

- Open the app in a browser to view the calendar.
- Use the UI to add reminders for specific days and times (see app controls in the page).
